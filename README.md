## Installation

1. Install Visual Studio Code 1.40.0 or higher
2. Launch Code
3. From the command palette `Ctrl`-`Shift`-`P` (Windows, Linux) or `Cmd`-`Shift`-`P` (OSX)
4. Select `Install Extension`
5. Type `NestJS Code Generator` and press enter
6. Install
